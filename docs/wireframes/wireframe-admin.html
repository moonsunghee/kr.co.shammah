■ 로그인 명세

  • 아이디/비밀번호 입력 → localStorage 또는 하드코딩 인증 (MVP)
  • 로그인 성공 시 admin 대시보드로 이동
  • 인증 실패 시 에러 메시지 노출
  • 세션: sessionStorage에 로그인 상태 저장
  • 모든 관리자 페이지 진입 시 인증 체크 → 미인증 시 로그인으로 리다이렉트
