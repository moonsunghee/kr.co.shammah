## 다운로드 방법

1. 위 URL을 브라우저에서 열기
2. 검색 결과 페이지인 경우 마음에 드는 사진 선택
3. **무료 다운로드** 버튼 클릭 (Pixabay 계정 필요)
4. 크기 선택 → **1280px** 또는 **640px** 권장
5. 다운로드 후 파일명을 위 표의 파일명으로 변경
6. `html/images/ai-literacy/` 폴더에 저장

---

## 업로드 후 동작

- 이미지 파일이 존재하면 → **사진이 카드 배경으로 표시**
- 이미지 파일이 없으면 → **카드별 그라디언트 컬러로 자동 대체**

사진 업로드 후 FTP로 `images/ai-literacy/` 폴더에 올리면 바로 반영됩니다.
